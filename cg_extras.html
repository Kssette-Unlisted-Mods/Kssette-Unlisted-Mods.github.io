Cassette Girl Extras Collection

Extra things for CG V-Slice Port

A small mod that adds different community levels and charts about Cassette Girl. This mod features elements from others, which you will find credited in the metadata. Enjoy it!

Warning: Required Dependency

To use this collection, you must first download the VS Cassette Girl Port from GameBanana.

IMPORTANT:

The name of your base mod folder MUST be exactly vs_cassette_girl_v162 for these extras to work properly.
